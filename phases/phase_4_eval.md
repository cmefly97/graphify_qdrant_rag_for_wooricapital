# Phase 4 — 평가·튜닝 (Evaluation)

## 목표
정확도(특히 수치)·근거 적합성·환각률을 측정하고 튜닝한다.

## Tasks
- [ ] 골든셋 구성: 제공 예시 5 + 각 5~10 변형(동의어·축약·오타) = 50~100문항
- [ ] 평가 스크립트: 수치 Exact-match, 출처 정확도, 환각률, "모름" 적정성
- [ ] 회귀 테스트(인덱스/프롬프트 변경 시 자동 재실행)
- [ ] 라우팅/프롬프트/재랭킹 파라미터 튜닝
- [ ] 결과 리포트 + 한계·후속 과제 정리

## Outputs (산출물 — complete 시 존재 검증)
- `eval/goldenset.jsonl`
- `eval/run_eval.py`
- `eval/report.md`

## Exit Criteria
- 골든셋에서 수치형 오답률 0%(또는 "확인 필요" 적절 표기).
- 환각률·출처 정확도 목표치 충족.
- 평가 리포트 산출.
