# QUERY_TRACE_EXAMPLE — 질의 처리 과정 상세 (실측)

> 실제 코드(`app/`) 경로를 그 질의로 실행해 각 단계의 값을 그대로 기록한 문서.
> 목적: "답이 어떻게 나오는지"를 재현 가능하게 남겨, 디버깅·설명·검증의 기준으로 사용.

## 예시 질의
```
금리등급 2등급일 때 적용 될 수 있는 최저 금리가 궁금해
```

## 최종 답변
```
금리등급 2등급시 21.0% G/L금리이며 내국인 네고 1%(N1) 적용됩니다.
추가로 거점장 네고 11.0%, 증빙 네고 15.0%, HJ 네고 18.0%가 가능합니다.
```
- mode: **table** · 신뢰도: **high** · route: **numeric**
- 출처: `9_샘플_중고승용 상품운영기준.pdf` (유효일자 2026-03-01)

---

## 단계별 처리 (실측 트레이스)

| # | 단계 | 모듈 | 소요 | 핵심 동작 |
|---|------|------|------|-----------|
| ① | 질의 라우팅 | `app/router.py` | 0.0ms | "금리·등급·숫자" → 수치 힌트 감지, 규칙 힌트 없음 → **numeric** |
| ② | 쿼리 확장 | `app/expander.py` | 0.0ms | 동의어 4종 생성(금리→G/L금리·rate·이자율). 설명형 검색 보조용 |
| ③ | **구조화 테이블 조회** | `app/tables_lookup.py` | 0.9ms | **여기서 답 확정** (아래 상세) |
| ④ | 벡터 유사도 검색 | `app/retriever.py` | 562.6ms | Octen 임베딩으로 상위 5청크 — 근거 패널용(지연 대부분) |
| ⑤ | 그래프 탐색 | `app/retriever.py` | 0.9ms | 시작문서 3 → 연결노드 5개(금리등급/G-L금리/NEGO 등) |
| ⑥ | 재랭킹 + 출처부착 | `app/reranker.py` | 0.0ms | 테이블 결과 최상위 + 벡터 청크 = 근거 6건 정렬 |
| ⑦ | 답변 생성 | `app/generator.py` | 0.0ms | table_result 존재 → **LLM 미호출**, 테이블 문장 그대로 반환 |

> 전체 지연의 대부분은 ④ 임베딩 API 호출. 답변 생성 자체는 0ms(LLM 미사용).

---

## ③ 테이블 조회 상세 — 숫자의 출처

`tables_lookup.lookup()` 은 순서대로 시도한다:
1. `lookup_residual_group()` → None ('잔가' 키워드 없음)
2. `lookup_condition()` → None (개월/듀얼/슬라이딩/R판정 아님)
3. **`lookup_rate()` → 매칭**

`lookup_rate()` 파싱·조회 결과(실측):
- 정규식 파싱: 등급=`2`, 개월=`없음`, NICE점수=`없음`(질의에 "___점" 없음)
- 고객유형: "외국인" 미포함 → **내국인**
- `rate_grade` 조회:
  ```
  내국인 · 등급2 · 12개월 → gl_rate = 21.0%   (source: 9_중고승용 pdf, 2026-03-01)
  ```
- `nego_rule` 조회 (grade_from ≤ 2 ≤ grade_to, 내국인):
  ```
  internal  nice_condition=1%(N1)  nego_rate=1%(N1)
  거점장     nego_rate=11.0%
  증빙       nego_rate=15.0%
  HJ        nego_rate=18.0%
  ```

### 답변 문장 조립 규칙
- NICE 점수가 **있으면**: "…이며 NICE n등급이므로 {internal nego} 네고 적용됩니다"
- NICE 점수가 **없으면**(이번 경우): "…이며 **내국인 네고 1%(N1)** 적용됩니다"
- 이어서 거점장/증빙/HJ 네고율을 "추가로 … 가능합니다"로 부착

→ **모든 수치가 tables.db 조회값**이며 LLM 생성이 아님 (CLAUDE.md §1.1 수치 환각 금지 준수).

---

## 이 경로가 주는 보장
- **정확성**: 21.0% 등 숫자는 운영기준 원본에서 추출된 테이블 값.
- **속도**: 수치형은 테이블 직조회 → 생성 0ms.
- **환각 0**: LLM이 숫자를 만들 여지가 없음.
- **추적성**: 답변 출처(테이블) + 우측 패널의 벡터 청크·graphify 노드로 근거 확인.

## 재현 방법
```python
from app.api import answer_query
r = answer_query("금리등급 2등급일 때 적용 될 수 있는 최저 금리가 궁금해", with_trace=True)
print(r["answer"]); print(r["trace"]); print(r["sources"])
```
> 설명형(테이블 미매칭) 질의는 ③에서 None → ④⑤ 검색 결과로 ⑦ LLM(HCX-30B / hcx-agent-05) 생성 경로를 탄다. 근거 없으면 "규정에 명시되어 있지 않습니다".
